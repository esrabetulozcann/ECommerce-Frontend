# 🛍️ E-Commerce Frontend (Trendyol Clone)

This project is a **Trendyol clone e-commerce application** built with **React, HTML, and CSS**.  
It uses my custom-built **E-Commerce API** as the backend.  

The purpose of this project is to simulate a real-world shopping experience where users can **search for products, add them to favorites, manage their cart, and proceed to checkout and payment**.

---

## 🚀 Features
- 🔍 **Homepage**: Search bar, categories, banner area, product listing, and loading screen.  
- 👤 **Account Menu**: Orders, user profile, Trendyol assistant, and logout option.  
- ❤️ **Favorites**: Displays all products added to favorites by the user.  
- 📄 **Product Detail**: Product images, price, size selection, and add-to-cart functionality.  
- 🛒 **Cart**: Product list, order summary, shipping information, and checkout button.  
- 💳 **Payment Page**: Delivery address form, payment options (credit card or bank transfer).  
- ⚙️ **User Profile**: Update user information, change password, and enable two-factor authentication.

  # 🛍️ E-Commerce Frontend (Trendyol Clone)

Bu proje, **React, HTML ve CSS** kullanılarak geliştirilmiş bir **Trendyol klonu e-ticaret uygulamasıdır**.  
Arka planda, kendi geliştirdiğim **E-Commerce API** kullanılmıştır.  

Projenin amacı, kullanıcıların **ürünleri arayabilmesi, favorilere ekleyebilmesi, sepete ürün atabilmesi ve ödeme adımlarına kadar ilerleyebilmesidir**.

---

## 🚀 Özellikler 
- 🔍 **Anasayfa**: Arama çubuğu, kategoriler, banner alanı, ürün listesi ve loading ekranı.  
- 👤 **Hesabım**: Siparişlerim, kullanıcı bilgileri, Trendyol asistanı, çıkış yap.  
- ❤️ **Favoriler**: Kullanıcıların favoriye eklediği ürünleri görüntüleme.  
- 📄 **Ürün Detayı**: Ürün görselleri, fiyat, beden seçme ve sepete ekleme.  
- 🛒 **Sepet**: Ürün listesi, sipariş özeti, kargo bilgisi, alışverişi tamamla butonu.  
- 💳 **Ödeme**: Teslimat adresi, kredi kartı veya havale/EFT ile ödeme seçenekleri.  
- ⚙️ **Kullanıcı Bilgileri**: Kullanıcı profili düzenleme, şifre güncelleme, iki adımlı doğrulama.
